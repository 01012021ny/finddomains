# Abandoned Domain Finder

Инструмент для поиска брошенных доменов, на которых раньше были работающие сайты (блоги, интернет-магазины, SaaS-сервисы и др.), с целью их покупки и восстановления.

## Что делает

- **Проверяет историю домена** через Wayback Machine CDX API (количество снапшотов, возраст)
- **Определяет тип сайта** (блог, магазин, SaaS, форум, цифровые товары) по HTML-сигнатурам
- **Проверяет доступность** домена через WHOIS
- **Оценивает бэклинки** через CommonCrawl
- **Рассчитывает скор ценности** (0-100) на основе всех метрик
- **Генерирует домены по нишам** (digital products, SaaS, ecommerce и др.)
- **Ищет в архиве** по ключевым словам

## Быстрый старт

```bash
pip install -r requirements.txt
python app.py
```

Открыть http://localhost:5000

## Docker

```bash
docker build -t domain-finder .
docker run -p 5000:5000 domain-finder
```

## API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/check` | Проверить один домен `{"domain": "example.com"}` |
| POST | `/api/check-batch` | Проверить список `{"domains": ["a.com", "b.com"]}` |
| POST | `/api/search-archive` | Поиск по ключевому слову `{"keyword": "tech-blog"}` |
| POST | `/api/generate` | Генерация по нише `{"niche": "saas"}` |
| GET | `/api/niches` | Список доступных ниш |

## Стратегия использования

1. **Генерируй домены по нише** — выбери нишу, получи список потенциальных доменов
2. **Ищи в архиве** — используй ключевые слова вроде "shopify-store", "wordpress-blog"
3. **Проверяй найденные** — анализируй историю, тип контента, доступность
4. **Фильтруй по скору** — домены с 60+ баллов — лучшие кандидаты
5. **Покупай и восстанавливай** — используй archive.org для восстановления контента
